This allows you to display simple text, can be configured like so:

```yaml
- greeting:
    text_size: xl
    text: Greeting Text
```

Valid text sizes are `4xl`, `3xl`, `2xl`, `xl`, `md`, `sm`, `xs`.

*Added in v0.3.48*
