title: Overseerr
description: Overseerr Widget Configuration
---

Find your API key under `Settings > General`.

Allowed fields: `["pending", "approved", "available", "processing"]`.

```yaml
widget:
    type: overseerr
    url: http://overseerr.host.or.ip
    key: apikeyapikeyapikeyapikeyapikey
```
