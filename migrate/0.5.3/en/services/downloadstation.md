Note: the widget is not compatible with 2FA.

Allowed fields: `["leech", "download", "seed", "upload"]`.

```yaml
widget:
    type: downloadstation
    url: http://downloadstation.host.or.ip:port
    username: username
    password: password
```

*Added in v0.5.3*
