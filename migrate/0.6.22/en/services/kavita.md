Uses the same username and password used to login from the web.

Allowed fields: `["seriesCount", "totalFiles"]`.

```yaml
widget:
    type: kavita
    url: http://kavita.host.or.ip:port
    username: username
    password: password
```

*Added in v0.6.22*
