---
title: Jellyseerr
description: Jellyseerr Widget Configuration
---

Find your API key under `Settings > General > API Key`.

Allowed fields: `["pending", "approved", "available"]`.

```yaml
widget:
    type: jellyseerr
    url: http://jellyseerr.host.or.ip
    key: apikeyapikeyapikeyapikeyapikey
```
